# Run Orderbook Instructions

### Clone from GitHub

```
git clone https://github.com/im-yifanluo/rust-final-project.git
```

### Run via Console Commands

To start running the program, enter the following in the console within the project's root directory:
```
cargo run
```

After the project is running, you should see:
```
Welcome to the Order Book designed and implemented by Suhaan Khan and Yifan Luo!

===============================
FMLT Order Book
===============================
What would you like to do today?
 1) Buy 
 2) Sell 
 3) View Live Bids 
 4) View Live Asks 
 5) Past Orders 
 6) Past Trades 
 7) Quit
```
Enter any integer from 1 to 7 and press enter to perform an action.
